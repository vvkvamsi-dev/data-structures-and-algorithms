## Pattern
Array Manipulation / Index Mapping

## Approach Used
Created a new array and placed:
- nums[i] at index 2*i
- nums[i+n] at index 2*i+1

## Time Complexity
O(n)

## Space Complexity
O(n)

## Other Approach
Bit Manipulation can achieve O(1) extra space by 
encoding two numbers in one array element, 
but I have not studied bit manipulation yet.
